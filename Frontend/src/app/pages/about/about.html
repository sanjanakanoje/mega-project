About Smart Textile System

ABOUT PAGE WORKING

The Smart Textile Management System is designed to manage textile samples, track fabric details, and improve workflow in textile industries.

Our Objective

Our goal is to simplify textile data management, reduce manual work, and improve accuracy in sample tracking.

Features

  • ✔ Add and manage textile samples
  • ✔ User authentication (Login/Register)
  • ✔ Profile management
  • ✔ Easy navigation dashboard

Technology Used

  • Angular (Frontend)
  • Node.js (Backend)
  • MySQL (Database)
